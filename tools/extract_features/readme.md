# Параллельное извлечение метрик из текста
Скрипт загружает DataFrame  файла, разбивает указанную текстовую колонку на чанки указанного размера и параллельно обрабатывает их, сохраняя результаты в указанную выходную директорию.

## Параметры и дефолтные значения

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `input_path` | Путь к DataFrame | Обязательный параметр |
| `text_column` | Название колонки с текстом | Обязательный параметр |
| `--output-dir` | Директория для сохранения результатов | `.` (текущая директория) |
| `--tmp-dir` | Директория для временных файлов чанков | `./tmp` |
| `--chunk-size` | Количество строк в одном чанке | `250` |
| `--max-workers` | Максимальное количество параллельных процессов | число CPU (`os.cpu_count()`) |

## Пример использования

```sh
# pwd - tools/extract_features

# Базовый запуск
python3 main.py ../../data/rusid/processed/rus_idiolect_clean.parquet text

# Запуск с указанием размера чанка и количества процессов
python3 main.py ../../data/rusid/processed/rus_idiolect_clean.parquet text \
    --chunk-size 500 \
    --max-workers 4

# Запуск с указанием директорий для сохранения
python3 main.py ../../data/rusid/processed/rus_idiolect_clean.parquet text \
    --output-dir ./results \
    --tmp-dir ./temp_chunks
```
