# TextMetrics

## Taggers (Тэггеры)
Тэггер - модуль, который находит и размечает сущности в исходном тексте.

**Требования к тэггеру**. Каждый тэггер должен содержать:
- Атрибут `TAG_NAME` - имя размечаемой сущности (строка)
- Функция `tag(text)` - принимает строку, возвращает список найденных участков

### Важные замечания
- Порядок тэггеров имеет значение! Если тэггер выделяет сущность, диапазон которой уже занят другой разметкой - сущность не будет добавлена.
- Все тэггеры должны находиться в директории `entities/taggers/`.  Эта папка использутеся `TaggerLoader` по умолчанию
- Загрузка по имени: `url` → `taggers/url.py`

## Metrics (Метрики)
Метрика - модуль, который рассчитывает характеристики текста на основе контекста (исходный текст + размеченные сущности).

**Требования к метрике**:
- Атрибут `METRIC_NAME` - имя метрики (обязателен для одиночных значений. Если возвращается словарь, то используются ключи словаря)
- Атрибут `METRIC_GROUP` - группа метрики (опционально, по умолчанию `common`)
- Функция `extract(context)` - принимает объект контекста, возвращает словарь с результатами

Метрики расположены в директориях `metrics`. Путь к папкам с метриками указывается функции `load_by_directories` в качестве параметра.

## Пример использования
```python
from text_metrics.processor import extract 

values = extract("Итак, не можешь не писать – не пиши. Усмири свой пыл любым способом, который тебе знаком лучше, чем мне...")

print(values)
```
